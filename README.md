# Synod

![Build and Test Status](https://github.com/filmil/synod/actions/workflows/build-test.yml/badge.svg)

Synod is a repository initialized for use with Bazel, Go, and gRPC.

## Features

- **Bazel Build System**: Configured with version 9.0.1.
- **Go Support**: Integrated using `rules_go` and `gazelle`.
- **Protocol Buffers & gRPC**: Configured for cross-platform gRPC services.
- **CI/CD**: Automated build and test using GitHub Actions.
- **Customizable Bazel Configuration**: Support for `user.bazelrc` via `.bazelrc`.
